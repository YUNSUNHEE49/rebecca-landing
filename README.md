# Arabic Landing Page

Simple static Arabic landing page prepared for Vercel.

## Files

- `landing-ar.html` - main landing page
- `index.html` - redirects root visitors to the Arabic page
- `vercel.json` - Vercel routing settings

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Go to Vercel and import the repository.
3. Keep the framework preset as `Other`.
4. Leave build command and output directory empty.
5. Deploy.

You can also deploy with the Vercel CLI:

```bash
npx vercel --prod
```
